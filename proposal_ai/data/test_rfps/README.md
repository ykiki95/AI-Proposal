# test_rfps/

AI 제안서 생성을 테스트하기 위한 **RFP(제안요청서) PDF 파일**을 이 폴더에 넣어주세요.

## 용도

- 새 제안서를 생성할 때 입력으로 사용하는 RFP 문서입니다.
- 개발·테스트 단계에서 실제 RFP를 업로드하기 전 검증용으로 사용합니다.

## 파일 규칙

- **형식**: PDF (`.pdf`)
- **파일명 예시**: `2024_스마트시티_RFP_테스트.pdf`, `샘플_클라우드전환_RFP.pdf`
- 실제 운영 시에는 시스템 UI를 통해 RFP를 직접 업로드하게 됩니다.

## 주의사항

- 이 폴더의 `.pdf` 파일은 `.gitignore`에 의해 git에 커밋되지 않습니다.
- 기밀이 포함된 RFP는 반드시 보안 지침에 따라 관리하세요.
