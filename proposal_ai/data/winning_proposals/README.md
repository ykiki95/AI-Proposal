# winning_proposals/

수주에 성공한 제안서의 **PDF 파일**을 이 폴더에 넣어주세요.

## 용도

- RAG(검색 증강 생성) 시스템의 학습 데이터 소스로 사용됩니다.
- AI가 과거 수주 제안서의 내용·구조·표현 방식을 학습합니다.

## 파일 규칙

- **형식**: PDF (`.pdf`)
- **파일명 예시**: `2024_스마트팩토리_ABC사_수주.pdf`, `2023_클라우드전환_XYZ사.pdf`
- 파일명에 프로젝트명, 고객사, 연도를 포함하면 관리가 편리합니다.

## 주의사항

- 이 폴더의 `.pdf` 파일은 `.gitignore`에 의해 git에 커밋되지 않습니다.
- 보안 및 저작권 보호를 위해 실제 제안서 파일은 별도로 관리하세요.
- PPT 원본이 있는 경우 `../winning_proposals_pptx/` 폴더에 함께 보관하세요.
