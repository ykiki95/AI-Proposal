# winning_proposals_pptx/

수주에 성공한 제안서의 **PowerPoint 원본 파일**을 이 폴더에 넣어주세요.

## 용도

- 슬라이드 레이아웃, 색상 팔레트, 폰트 등 **디자인 패턴 추출**에 사용됩니다.
- AI가 제안서의 시각적 구조와 표현 방식을 학습합니다.
- PDF에서 손실된 편집 가능한 텍스트와 도형 정보를 보완합니다.

## 파일 규칙

- **형식**: PowerPoint (`.pptx`, `.ppt`)
- **파일명 예시**: `2024_스마트팩토리_ABC사_수주.pptx`, `2023_클라우드전환_XYZ사.pptx`
- `winning_proposals/` 폴더의 PDF 파일과 동일한 이름을 사용하면 매칭이 쉽습니다.

## 주의사항

- 이 폴더의 `.pptx`, `.ppt` 파일은 `.gitignore`에 의해 git에 커밋되지 않습니다.
- 보안 및 저작권 보호를 위해 실제 제안서 파일은 별도로 관리하세요.
